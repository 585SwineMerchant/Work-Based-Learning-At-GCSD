Accessibility

Accessibility Statement

The staged site should be usable by keyboard, readable on small screens, and available in alternate formats when requested through district channels.

Access

Public commitment

GCSD should provide public pathway information in formats that students, families, staff, and community members can use. This staged site uses semantic headings, landmarks, keyboard-operable navigation, visible focus styles, and responsive layouts.

This statement does not claim formal WCAG conformance. A full accessibility audit should be completed before live publication.

Before live use

Known limitations

  • Some resource routes are not yet assigned to monitored district owners.
  • No public forms or downloads should go live until content, accessibility, privacy, and ownership review are complete.
  • Any linked documents should be checked for headings, reading order, tables, contrast, and alternate text before publication.
Request route

Accessible alternatives

Until a dedicated monitored route is assigned, request accessible alternatives through existing district or school communication channels.
