# Table Covering Concept

A design concept for the presentation table's covering/tablecloth, written so
it can be handed to a print shop or a home-printing/DIY setup without further
translation.

> Course project draft — proposed program, not official Greece Central School
> District policy or branding.

## Title / Tagline Placement

- **Front-facing panel** (the edge visitors see approaching the table):
  large centered title **"Greece Culinary Work-Based Learning"** with the
  tagline **"From Classroom Kitchen to Career"** directly beneath it in a
  smaller, complementary weight.
- Keep the front panel to title + tagline only — no body copy. It needs to
  be readable from several feet away, before a visitor has stopped walking.
- If the covering has side panels, a single supporting line may repeat there
  (e.g., "Course Project Draft — Proposed Program"), small and unobtrusive.

## Colors / Style

Use the same palette already established on the website, so the table reads
as one system with the printed materials and the slideshow:

- **Primary:** deep forest green (`#2E4630`) — main covering color or title
  background
- **Accent:** warm terracotta/gold (`#A5601C`) — tagline, dividing lines, or
  small accents
- **Neutral:** warm off-white/cream (`#FAF8F4`) — title text on the green
  field, or as a secondary panel color

Overall style: warm, clean, modern CTE/career-pathway — professional rather
than "restaurant-themed." Avoid checkerboard/chef-hat kitsch; this is a
career-pathway table, not a diner display.

## No Official Logo

Do not use an official Greece Central School District logo, seal, or
wordmark on the covering unless a real, approved logo file is provided by
the district. In its place, use a simple text-based mark — the same "GC"
monogram treatment used in the website header — or the title text alone,
cleanly set.

## Suggested Visual Elements

- A thin horizontal divider line in the accent color between the title and
  tagline
- Optional: a subtle, single-color line-art icon (a whisk, toque, or simple
  plate outline) placed off to one side — small and understated, not a
  background pattern
- Optional: a small QR code printed directly on a front table-runner strip,
  as a backup in case the display board's QR code is hard to see from a
  distance (label: "Scan to explore")

## Accessibility & Legibility Considerations

- Maintain strong contrast: cream/white text on the deep green field (this
  combination is already verified for contrast on the website) rather than
  low-contrast color-on-color text
- Use a clean sans-serif typeface at a size readable from at least 6–8 feet
  away for the title
- Avoid placing text over busy patterns or photos — solid color fields keep
  text legible under variable event lighting
- If a QR code is printed on the covering, keep a printed short URL directly
  beneath it as a non-scanning backup (see `qr-code-instructions.md`)

## How the Covering Supports the Display

The covering is the visual base layer the rest of the table sits on top of:
it establishes the color system and title treatment that the display board,
flyers, and slideshow all repeat, so a visitor's eye reads the whole table as
one coordinated presentation rather than a pile of separate handouts. Because
the covering carries the title and tagline, the display board itself is
freed up to start directly with content (Panel 1 in `display-board-text.md`)
rather than repeating the title.
