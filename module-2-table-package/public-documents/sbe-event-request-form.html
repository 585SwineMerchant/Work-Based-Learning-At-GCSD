SBE Event Request Form

Culinary Pathway · Greece Central School District

Use this form to request a student-run School-Based Enterprise (SBE) event — catering, a pop-up event, a bakery sale, or another student-prepared food experience. Submitting this form starts the planning conversation with the culinary instructor/coordinator; it does not guarantee the event will be scheduled.

This is a sample form created as part of a graduate course project draft (SUNY Oswego, CTE 566). It describes a proposed program and is not an official Greece Central School District communication. Final event request procedures, approval workflow, and food-sale/fundraising rules would be determined by district approval and would need to follow Greece Central School District policy and applicable health/safety requirements.
