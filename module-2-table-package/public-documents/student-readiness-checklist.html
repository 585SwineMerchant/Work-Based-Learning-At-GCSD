Because production and service roles vary widely — prep, plating, service, dishwashing, front-of-house — this checklist is designed to support students with different strengths, needs, 504 plans, or IEPs through role rotation rather than a single fixed standard.

This is a draft self-assessment tool created as part of a graduate course project (SUNY Oswego, CTE 566). It describes a proposed program and is not an official Greece Central School District communication. Any formal readiness or evaluation process would be determined by district approval and would need to follow applicable district policy and New York State requirements.
