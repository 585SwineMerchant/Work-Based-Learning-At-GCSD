# Module 2 Presentation Table Rationale

**Greece Culinary Work-Based Learning — From Classroom Kitchen to Career**

> Course project draft. This document supports a graduate coursework assignment (SUNY Oswego, CTE 566: Implementing and Evaluating Work-Based Learning Programs) and describes a proposed program. It is not an official Greece Central School District communication and does not represent official district policy.

## Assignment Purpose

The Module 2 Presentation Table Project asks the WBL coordinator to step into a public-facing role: promoting a proposed Work-Based Learning program at a community-facing event and communicating its purpose, value, and impact in a compact, visitor-friendly format. This package assembles the four required table components — a display board, a table covering, supplementary literature, and an attention getter — around a single consistent identity: **Greece Culinary Work-Based Learning, From Classroom Kitchen to Career**. The same table is designed to be reusable, with light updates, for the Module 5 Board of Education presentation.

## Intended Audience

The table is designed to read clearly to five different visitor types who may stop by with very different questions and very little time:

- **Students** — "What would I actually do?"
- **Families** — "Is this safe, and what's expected of my student?"
- **Employers and community partners** — "What would it look like to get involved?"
- **School and district staff / administrators** — "Is this organized, standards-aligned, and worth supporting?"
- **General community members** — "What is this program, in one sentence?"

Because a table conversation is often 30–90 seconds, every printed and displayed element is written to work at a glance, with the website available for anyone who wants more depth.

## Why a QR-Accessible Website Hub Strengthens the Table

A physical table can only hold so much text before it becomes unreadable. Rather than trying to print every detail, this package treats the [Greece Culinary Work-Based Learning website](https://585swinemerchant.github.io/Work-Based-Learning-At-GCSD/) as the **central hub** and the table materials as a **short, high-quality index** into it. Every flyer, the display board, and the attention-getter slideshow all point to the same QR code and short URL, so a visitor who wants to go deeper — FAQs, forms, the full student pathway, evaluation framework — always lands in the same well-organized place instead of a scattered set of one-off handouts. This also means the table materials stay accurate over time: as the site is updated, the table's QR-linked content updates with it, without reprinting anything.

## How the Table Promotes WBL

The table is built to teach the core idea of Work-Based Learning in under a minute: WBL is **structured, supervised, standards-aligned career preparation** — not simply students being placed in jobs. Every component reinforces the same four-part structure (classroom → lab → school-based enterprise → employer/community partnership) so a visitor leaves with a consistent mental model, regardless of which piece of literature they picked up or which panel of the display board caught their eye.

## How the Table Connects Existing Culinary Strengths to Expanded WBL

A central message of this package is that the Culinary Pathway **is not starting from zero**. The display board, flyers, and slideshow all name concrete, already-happening activity — ServSafe certification, restaurant visits, guest chef speakers, in-house catering for student and faculty events, curriculum-tied menu planning and costing, an internal purchase-order/accountability process, scratch production, packaging and service, and business-class collaboration on pricing and marketing materials. The proposed WBL system is framed throughout as a way to **formalize and expand these existing pathway strengths** — documenting student growth, connecting internal culinary experiences to external employer and community partnerships, and using school-based enterprise as a readiness bridge — rather than inventing a new program out of nothing. This framing is more accurate to the program's actual starting point and more persuasive to employers and administrators who can already see evidence of quality.

## How Each Required Component Is Included

| Required component | How it's met in this package |
|---|---|
| Display board | `display-board-text.md` — eight concise panels covering what WBL is, existing strengths, the student pathway, school-based enterprise, employer partnership, the QR hub, evaluation, and a closing call to scan |
| Table covering | `table-covering-concept.md` — a professional covering concept using the site's color palette and "From Classroom Kitchen to Career" identity, with no district logo unless one is later approved |
| Supplementary literature | `student-family-flyer-copy.md` and `employer-partner-flyer-copy.md` — two audience-specific one-page flyers, plus the website itself as the deeper reference layer |
| Attention getter | `attention-getter-slideshow-outline.md` — a short, simple, loopable slideshow for a tablet or laptop at the table |

## Supporting the Future Module 5 Board Presentation

Because this table package and the website share one identity, one color system, and one set of talking points, the same assets are designed to be lightly adapted — rather than rebuilt — into the Module 5 Board of Education presentation. The Evaluation & Impact framing already on the site (and echoed on the display board) is written to give a future Board presentation something concrete to point to: what would be measured, how the advisory board would be involved, and how findings would be communicated back to families, employers, and district leadership.
